# La Revolución del Machine Learning en Medicina

Este repositorio contiene el código fuente (RMarkdown) de un manual introductorio sobre Inteligencia Artificial y Machine Learning, diseñado específicamente para estudiantes de pregrado en medicina.

El objetivo de este proyecto es explicar conceptos matemáticos complejos y algoritmos (como Regresión Logística, Redes Neuronales Convolucionales y Transformers) utilizando analogías clínicas y metáforas del día a día en un hospital.
